Web Development

Web development is a broad term for the work involved in developing a web site for the Internet or an intranet.

  • Front-end development: This involves creating the visual elements of a website that users interact with directly.
  • Back-end development: This focuses on the server-side of web applications, including databases and server logic.
  • Full-stack development: This encompasses both front-end and back-end development skills.

In this course, students will learn and be able to work as web developers.
